# Enterprise Knowledge Platform

AI-powered enterprise knowledge platform for document intelligence, semantic search, knowledge graph generation, and intelligent recommendations.

## Features

- Document processing
- Semantic search
- Knowledge graph generation
- Intelligent recommendations
- Workflow orchestration
- FastAPI API
- Docker support

## Architecture

- Document Agent
- Search Agent
- Knowledge Graph Agent
- Recommendation Agent
- Knowledge Workflow

## Technology Stack

- Python
- FastAPI
- Docker
- Pytest
